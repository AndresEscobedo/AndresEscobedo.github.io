UP Dynamics

Proyecto de programación


Con el mercado de prótesis en México siendo un poco obsoleto en tecnología y con altos costos, el equipo de Panteras Dynamics se propuso crear una prótesis (rodilla) con tecnología de punta y costos accesibles. El reto es integrar movilidad completa de la prótesis al usuario, y para esto se hará uso de machine learning mediante la implementación de un microcontrolador y diversos sensores. El mayor reto ha sido la obtención de una base de datos para el entrenamiento del modelo y la interacción en tiempo real con actuadores para obtener la movilidad completa que se busca.

Keywords: "Python", "IA", "ML"
